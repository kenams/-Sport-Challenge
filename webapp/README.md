SportClash — web prototype

Dossier `webapp/` : petite webapp Vite + React reproduisant l'interface (pages + onglets).

Quick start:

```bash
cd webapp
npm install
npm run dev
```

Visiter http://localhost:5173

Fichiers créés:
- src/App.jsx, src/main.jsx
- src/pages/* (Home, Sports, Live, Bets, Coach)
- src/components/NavTabs.jsx
- src/styles.css

Prochaines étapes possibles:
- Affiner responsive et UX
- Intégrer contenu dynamique depuis Supabase
- Exporter assets et lier au projet mobile
